# TMD Editor

A simple markdown editor built on top of ProseMirror.

## Roadmap

### v0.0.1

- Target Release: Q1 2023
- Overview: Basic syntax support
   - Horizontal Rule
   - Code Block
   - Bold, Italic, Strikethrough
   - Link
   - Inline Code
   - Sub, Sup
   - Special Characters (e.g. &copy)
   - Image
